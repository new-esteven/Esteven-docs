control.gd

func load_data(path): loads and parses a json file from the provided path and returns the data as a dictionary. used for save files, battle data, level scaling and pretty much anything stored in json.
func load_character(number): shows the selected party member, updates their hp and skill names in the ui and hides the previously selected character. used whenever the player changes turns.
func shadow_turn(): handles the enemy phase. every shadow attacks a random party member, updates the battle log and checks if the battle has ended. if all enemies are defeated the battle ends immediately.
func _ready(): loads the battle data and stat tables, gathers all characters and enemies from the scene tree and initializes their stats. once everything is loaded it selects the first party member and starts the battle.
func change_turn(): advances to the next party member. if every ally has already acted it starts the enemy turn before returning control to the player.
func attack(shadow, multiplier, base_attack, base_defense, skill_power, is_physical_attack, luck, skill): calculates damage using the battle formula, applies critical hits and random variance, updates the target hp and displays the damage dealt on screen.
func play_sound(character): plays the attack sound effect associated with the specified character and temporarily lowers the background audio volume while it plays.
func action(skill, character, sword, shadowlife, shadow): executes an action during battle. this can be either a normal weapon attack or a skill attack. after applying damage it updates the battle log and checks if the enemy has been defeated.
func end_battle(won, character, is_shadow_turn): handles the end of combat. if the player wins it displays the victory screen and will later be responsible for rewards and scene changes. losing currently has not been implemented yet.
func hide_menu(): hides all battle menus and skill selection interfaces.
func open_persona_menu(): opens the skill selection menu and switches the current menu state to persona mode.
func open_general_menu(): opens the main battle menu and switches the current menu state back to general mode.
func _process(delta): handles all battle input every frame. manages menu navigation, skill selection, attacks, turn progression and returning between menus while preventing input during enemy actions.
